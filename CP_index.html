About

How does the web works?

The WEB is a set of networked machines communicating using a common language, it works in client / server sides, that is mean that there are machines called servers which offer resources and machines called clients which use these resources.
Those resources are for example, HTML pages, images, XML files (extensible Markup Language) or programs (PHP, Java, ASP.NET, Python, Perl, etc.) responsible for generating them on demand. The client accesses the resources using the HTTP communication protocol.
At the server level, in addition to the web server, we can have a data server that will host the Database Management System (DBMS). And to access it, we use the universal database query language SQL.

What do you need to be a web developer?

Anyone can become a web developer. You don’t need to be a tech wizard or possess a never-ending list of formal qualifications; as long as you’re passionate about the field and willing to learn, a career in web development is well within your reach.

Before you can call yourself a web developer and start applying for jobs, you’ll need to master the following skills:

  • Languages: Like HTML, CSS and JavaScript
  • Libraries and Frameworks: Like Bootstrap and jQuery ..
  • Git and GitHub
  • Optional: Design Software Sketch and Photoshop

Of course, this is just the tip of the iceberg, there are loads more languages, tools and platforms that web developers use in their day-to-day work. However, it’s important to start with the basics and walk before you can run!

Tutorial

What is HTML ?

L'HTML est un langage informatique utilisé sur l'internet. Ce langage est utilisé pour créer des pages web. L'acronyme signifie HyperText Markup Language, ce qui signifie en français "langage de balisage d'hypertexte". Cette signification porte bien son nom puisqu'effectivement ce langage permet de réaliser de l'hypertexte à base d'une structure de balisage.

What is CSS ?

Le terme CSS est l'acronyme anglais de Cascading Style Sheets qui peut se traduire par "feuilles de style en cascade". Le CSS est un langage informatique utilisé sur l'internet pour mettre en forme les fichiers HTML ou XML. Ainsi, les feuilles de style, aussi appelé les fichiers CSS, comprennent du code qui permet de gérer le design d'une page en HTML.

What is JS ?

JavaScript désigne un langage de développement informatique, et plus précisément un langage de script orienté objet. On le retrouve principalement dans les pages Internet. Il permet, entre autres, d'introduire sur une page web ou HTML des petites animations ou des effets.

What is ReactJS ?

React est une bibliothèque JavaScript libre développée par Facebook depuis 2013. Le but principal de cette bibliothèque est de faciliter la création d'application web monopage, via la création de composants dépendant d'un état et générant une page HTML à chaque changement d'état.

What is NodeJs ?

Node.js est une plateforme logicielle libre en JavaScript orientée vers les applications réseau événementielles hautement concurrentes qui doivent pouvoir monter en charge. Elle utilise la machine virtuelle V8, la librairie libuv pour sa boucle d'évènements, et implémente sous licence MIT les spécifications CommonJS.
